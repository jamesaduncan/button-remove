Button-remove example

Button-remove lets you declaratively add buttons to HTML documents that remove elements from the document.

To use button-remove, first of all include the script in the body of your document. 

<script type="module" src="https://jamesaduncan.github.io/button-remove/index.mjs"></script>

Then add the button to your document, providing a remove attribute with a removal method type, and a selector as the target.

Clicking the button removes this

It's also possible to remove elements using the closest removal type. Clicking this button will remove this paragraph.
